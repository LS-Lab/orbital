Contains template interfaces for logic functors like Predicates and Functions that can be used for generalized call-back.

The orbital.logic.functor.Functor interface is the basic interface tagging all sub interfaces that define functors. Instances of implementing classes can be used as a call-back through this common interface. Predicates are special Functors returning a boolean (success or failure), while Functions are Functors returning arbitrary objects (return value).

Note that this package contains templates. 

Common logic functor interfaces:

  • orbital.logic.functor.Functor
    • Functions
      • VoidFunction<B>: f:Ø→B
      • Function<A,B> (unary): f:A→B;x↦f(x)
      • BinaryFunction<A1,A2,B>: f:A1×A2→B;(x,y)↦f(x,y)
    • Predicates
      • VoidPredicate: P⊆Ø
      • Predicate<A> (unary): P⊆A
      • BinaryPredicate<A1,A2>: P⊆A1×A2
The class orbital.logic.functor.Functionals defines several useful static methods for composing or converting functors and for applying them in various recursion schemes.
