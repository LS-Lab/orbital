Call the Game applet within a html page per

<applet code="Game.class" width="250" height="250"> <param name="gameName" value="the Name of the Game"> <param name="gameRules" value="GameRulesImpl"> <param name="aIntelligence" value="1"> </applet>

To run this applet, either use the appletviewer, or a very modern Browser with Java Plug-in, or run it as an application by calling 

java Game gameRules=GameRulesImpl aIntelligence=1

To enable two player game, the aIntelligence setting must be reduced to zero. To enable computer auto play mode, the aIntelligence setting is increase to the total number of players.

In order to extend the basic game applet, you can write a sub class with the appropriate features. An example of a running game is kept in the examples\SChess directory. Its computer is kept all to simple though, to provide a very simple introduction of how to use it. Refer to the home page for additional examples.
