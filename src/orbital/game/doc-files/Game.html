Call the Game applet within a html page per

<applet code="orbital.game.GameView.class" width="250" height="250"> <param name="gameName" value="the Name of the Game"> <param name="gameRules" value="GameRulesImpl"> <param name="player-2" value="argumentForComputerSecondPlayer"> </applet>

To run this applet, either use the appletviewer, or a very modern Browser with Java Plug-in, or run it as an application by calling 

java Game gameRules=GameRulesImpl player-2=argumentForComputerSecondPlayer

To set league i to a human player, player-i must be set to the string null, or not set at all. Otherwise the argument is passed to the computer player.

In order to extend the basic game applet, you can write a sub class with the appropriate features. An example of a running game is kept in the examples/SChess directory. Its computer is kept very simplistic though, to provide a very simple introduction of how to use it. Refer to the Orbital library web site for additional examples.
