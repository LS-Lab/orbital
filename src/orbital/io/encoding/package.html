Provides character and typed data encoding and decoding filter streams.

Base-classes for filtered stream operations are DecodingReader and EncodingWriter. Their implementations can be accessed through the corresponding factory methods (see DecodingReader.getInstance(Reader, String) and EncodingWriter.getInstance(Writer, String)). There you can find an up to date list of the supported encodings as well. Some of them are:
Encoding types for DecodingReaders / EncodingWriters

default will perform default encoding leaving everything as is

Additionally several text based implementations of java.io.DataInput and java.io.DataOutput are provided and can be accessed through the corresponding factory methods (see DataReader.getInstance(Reader, String) and DataWriter.getInstance(Writer, String)). There you can find an up to date list of the supported encodings as well. Some of the encodings supported are:
