Provides arithmetic objects and mathematical utility classes.

Arithmetic Objects contained in this package are:
 

Types of Arithmetic Objects provided
Group Class value representation
scalar types Integer k ∈ Z
Rational p⁄q ∈ Q
Real a∈R
Complex a + i*b ∈ C
special Symbol "x"
Quotient ā ∈ M/~
vector space types Vector v ∈ An
Matrix M ∈ Am×n
Matrix3D M ∈ R4×4 (fast 3D transformations)
polynomial types Polynomial p ∈ R[X]
Here A denotes a set of arbitrary arithmetic objects. Z is the set of integers, Q is the set of rational numers, R is the set of real values, and C is the set of complex numbers. Then the type hierarchy of the modeled Arithmetic Objects realize the mathematical inclusion
Z ⊂ Q ⊂ R ⊂ C

Since most general arithmetic objects are modelled as interfaces to provide a maximum of flexibility, you need factory methods to create an arithmetic object value. The class Values is that central factory class which can create arithmetic object values from all kinds of primitive types.

Mathematical function types are provided in a sub package orbital.math.functional.
