Contains an architecture for strategies that select objects with regard to their weight.

(Not to be mixed up with the Design Pattern "Strategy").

Basic interface is the Selection, its implementations select from an Enumeration of Objects (or Collection) according to their weight. The Algorithm for weighting an Object is done by user defined implementations of the interface Weighting.

The strategy is selected by Evaluation that evaluates a Selection of all Objects already weighted by Evaluation. The Weighting is delegated to a user defined Weighting instance.

The Selection algorithms implemented are:

  • Selection.Selecting.best()
  • Selection.Selecting.min()
  • Selection.Selecting.max()
  • Selection.Selecting.sum()
