Contains implementations of some logics as well as a logic and mathematical expression parser.

Important classes are:

  • ClassicalLogic is an implementation of classical first-order logic. Once instantiated, it is best accessed through its interface orbital.logic.imp.Logic.
  • ModalLogic is an implementation of modal logic.
  • FuzzyLogic instead, is an implementation of fuzzy logic.
  • MathExpressionSyntax provides the syntax (and thus the parser) for mathematical expressions.
  • LogicParser is a general parser for logical expressions. It can be used by implementations of orbital.logic.imp.Logic.
