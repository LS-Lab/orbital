The pure Implementation Packages of the Orbital library are contained completely in the package orbital.moon.* and its sub packages.

Additional implementation-specific classes and execution tools, which are subject to
change without notice are kept here.

Important implementations available for Orbital library include:

  • LogicParser
  • Orbital Service Providers:
    • CORBA object-server
    • RMI object-server
    • Port-Server
    • Jini service provider
  • Orbital security provider
  • Assertion specification testing features
  • OrbitalControl Panel for easy administration of Orbital servers.
  • BreederControl Panel for simple control of genetic algorithm problems.
