Important Features

The Orbital library is a general extension to the java packages in the form of a foundation class library. It specifies several central concept classes (Core Packages), and whole class networks that contain extended components, generic abstractions, and additional services of reusable Java classes (Extension Packages).

Especially promising contents are logical functors and their mathematical counter part with numerical and symbolic mathematics, as well as functional evaluation and several algorithmic templates including search and planning.

These packages include:

  1. mathematical, numerical and statistical utilities, and conceptual interfaces for mathematics.
  2. Enhanced numerical mathematics, and even symbolic mathematical calculations are possible.
  3. Arithmetic objects with uniform mathematical operations like +, -, *, /, ^. This enables us to use a uniform calculation with various arithmetic implementation objects which include matrices, vectors and scalar types like integers, rational, real and complex numbers. Even full symbolic calculations and modulo calculations are now possible.
  4. Mathematical functions are generalized to fit the requirements of functional operations and allow symbolic derivation and sometimes integration. They now extend the basic logical functors instead to maximize compatibility. Additionally mathematical functions can be handled like arithmetic objects whenever this general view is of advantage.
  5. Several algorithmic templates are provided which is extremely helpful for rapid prototyping. Those algorithmic templates also include the most important (blind and heuristic) state space search algorithms. Even some implementations of Planning algorithms are provided.
  6. Term rewrite sytem facility achieved with fixed point operations of term unification and substitution.
  7. a generic framework for logical systems using extended logic with classical operators. Note that this is a framework in the sense of logic, but not in the sense of software architecture. Therefore it provides you with all logical operations without restricting the architecture of your application.
  8. algorithmic template frameworks including evolutionary (genetic) algorithm implementation
  9. A parser for logical and mathematical expressions including λ-operator notation of λ-calculus.
  10. functional evaluation with functor interfaces and function notation classes.
  11. a generic board game abstraction framework.
  12. graphical user interface (GUI) utilities and AWT-Extension Java Beans.
  13. lightweight 3D worlds graphics engine.
  14. I/O-Extensions.
  15. Scanner&Parser base classes for simple parsing jobs.
  16. some document encoding and formatting classes.
  17. concurrent synchronous and asynchronous connectors that supports a programming style with coroutines.
  18. And some more...

Release 0.9.5+ of Orbital library is capable of JAR indexing which is a new feature that will result in less consumption of network bandwidth for applets. See documentation of JDK 1.3+ for more details on how to use it.
