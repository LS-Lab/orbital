Notes on OrbitalControl

If you have problems accessing resource files or get an error message concerning a missing resource bundle or problems with Index of JDK1.3.0, consider unpacking the jar files with
jar xf ../lib/orbital-core.jar
jar xf ../lib/orbital-ext.jar
jar xf ../lib/orbital-services.jar
jar xf ../lib/moon.jar
before using OrbitalControl. Also make sure that you have started a valid environment for
  1. RMI
  2. CORBA
  3. Jini(tm)
You can then test whether it worked by starting the applications in directory examples/distributed.
